This is Synticore!

Build static websites with control, speed, and zero framework lock-in.

Synticore Website Compiler is built for developers who prefer writing HTML, CSS, and JavaScript directly. It automates repetitive build work, keeps output lean, and helps you ship static sites, landing pages, and documentation with less friction.

Why Build with Synticore?

Automated Build Process

Use a practical Gulp-powered task pipeline to automate work such as:

  • Minification of HTML, CSS, and JS
  • Image optimization, including optional watermarking
  • Font icon generation from SVG files
  • Auto-injection of metadata and table of contents
  • Real-time file watching and live reloading

User-Friendly GUI

Manage your project from an approachable CustomTkinter GUI that includes:

  • Directory management with recent project history
  • One-click task execution
  • Visual task categorization and descriptions

Seamless Packaging

Prepare deployment packages with:

  • Automatic removal of development-specific content
  • Compression into a deploy-ready ZIP file
  • Fine-tuned exclusion of unnecessary assets

Cross-Platform Compatibility

Run comfortably on Windows and Linux, with setup steps and tooling designed for consistent local builds.

Our Arsenal

(Core Capabilities)

Build Automation

  • Minify and optimize assets for production
  • JSON-driven project configuration
  • File watching that rebuilds on change

Dynamic Content Handling

  • Auto-generate breadcrumbs and structured navigation
  • Automated sitemap generation for better crawlability
  • Configurable depth for in-page Table of Contents (TOC)

Smart Asset Management

  • Intelligent caching to avoid redundant builds
  • File exclusion rules to keep your output clean
  • GraphicsMagick integration for advanced image handling

Easy Deployment

  • Ready-to-ship ZIP packages
  • Versioned and bundled output assets
  • Lightweight setup with minimal dependencies

The Synticore Strategy

(Workflow)

  1. Set up your project:
    Point Synticore at your working directory and validate project settings.

  2. Run tasks:
    Use the GUI to build, watch, and package with a single click.

  3. Deploy confidently:
    Generate a production-ready ZIP with development-only files excluded.

Requirements

(System Requirements)

  • OS: Windows, Linux
  • Dependencies: Node.js, Python
  • Supported File Types: .html, .css, .js, .scss, .png, .jpg, .svg, .mp4, and more.
